##プロジェクト名　Portfolio-site

## 1.サービスの概要
自身のスキルと開発実績を可視化するための動的ポートフォリオサイトです。

## 2.ターゲットユーザーと課題解決
- **想定ユーザー**: IT企業の採用担当者、エンジニア
- **ユーザーの課題**: ユーザーのスキルを具体的に把握しにくい
- **解決方法**: 実際に動作するWebアプリケーションを通じて、MVCモデルの理解やDB操作スキルを証明します

## 3.機能一覧
・プロフィール表示機能
・技術スキル一覧
・開発実績
・お問い合わせフォーム

## 4.使用技術
- **Languge**: Java24
- **Framework**: Spring Boot 3.x
- **Database**: H2 Database (開発用)
- **View**: Thymeleaf
- **Infrastructure**: GitHub Pages / Heroku / Render (予定)
- **Tool**: IntelliJ IDEA, Maven, Git,
